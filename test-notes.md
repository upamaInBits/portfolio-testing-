# Test Notes – Portfolio Website

## Testing Type
Exploratory Testing + UI Inspection + Responsive Testing

## Scope
- Navigation bar
- Hero section
- Education section
- Projects section
- Responsive layout behavior
- External links
- CSS inspection in Chrome DevTools

## Devices / Viewports Tested
- Desktop view
- iPad Pro (1024 × 1366)
- iPhone 14 Pro Max (430 × 932)

## Tools Used
- Chrome DevTools
- Elements panel
- Styles panel
- Device Toolbar (responsive mode)

## Observations

### Responsive Layout
- Navigation menu remains visible across tested viewports
- Navigation items wrap appropriately on mobile view
- Hero section content reflows from side-by-side layout to stacked layout on smaller screens
- Main content containers remain within the visible viewport
- Image scales appropriately across tested screen sizes

### UI Inspection
- CSS rules were inspected through the Styles panel
- Layout structure was reviewed through the Elements panel
- Page spacing and container behavior were validated across desktop, tablet, and mobile layouts

### Functional Checks
- Section content remains readable across tested viewports
- Visual hierarchy is preserved on smaller screens
- Core landing page content renders without visible breakage in tested views

## Summary
Responsive and UI inspection testing was performed using Chrome DevTools. Across the tested desktop, tablet, and mobile viewports, the portfolio homepage maintained readable layout structure, visible navigation, and appropriate content reflow. This inspection confirmed that the page adapts correctly to multiple screen sizes in the tested scenarios.
